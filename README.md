# Konoba Maslina
Kratki opis sto je web i o cemu govori (1-3 recenice).

## Live demo
https://demoprojektuwt.netlify.app

## Stranice
- index.html - Naslovnica
- jelovnik.html - Grid kartica jela i pica
- recepti.html - Recept tjedna s detaljima
- rezervacija.html - Forma za rezervaciju stola

## Tehnologije
HTML5, CSS3 (Flexbox, CSS varijable), JavaScript (vanilla)
Google Analytics 4, Netlify hosting
